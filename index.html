ks-log

Its a library for C++ Developers, which:

  • Logs faster, uses different streams for different threads
  • Increased customizability and ease to use
      can use following tags for logging:
    • %f - File Name
    • %l - Line Number
    • %t - Thread ID
    • %d - Date and Time
    • %s - Severety of The Log
    • %m - Message of Log
  • Can limit loging using severity constraints.
      The available severities are:
    • INFO
    • WARNING
    • DEBUG
    • ERROR
    • CRITICAL
